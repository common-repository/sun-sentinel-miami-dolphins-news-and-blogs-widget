== Description ==

Sidebar Widget for your blog. The latest Dolphins headlines plus frequent blog posts from Sun-Sentinel Dolphins writers Omar Kelly and Mike Berardino. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `sun-sentinel-dolphins.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag Miami Dolphins News and Blog Widget to your Sidebar

== Frequently Asked Questions ==
- How do I upload sun-sentinel-dolphins.php to the /wp-content/plugins/ directory?
- Use your FTP manager, browse to the /wp-content/plugins/ directory of your blog and upload sun-sentinel-dolphins.php to that folder.
